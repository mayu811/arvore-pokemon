# Como Executar a Aplicação

## 1. Clonar o Repositório

### Linux

```bash
git clone <URL_DO_REPOSITORIO>
cd <NOME_DO_REPOSITORIO>
```

### Windows (PowerShell ou Git Bash)

```bash
git clone <URL_DO_REPOSITORIO>
cd <NOME_DO_REPOSITORIO>
```

---

## 2. Compilar o Código

Certifique-se de possuir o compilador GCC instalado em sua máquina.

Execute o comando abaixo no terminal para compilar o arquivo `main.c`:

### Linux

```bash
gcc main.c -o arvoreb
```

### Windows

```bash
gcc main.c -o arvoreb.exe
```

Após a compilação, será gerado um arquivo executável (`arvoreb` no Linux ou `arvoreb.exe` no Windows).

---

## 3. Executar a Aplicação

### Linux

```bash
./arvoreb
```

### Windows

```bash
arvoreb.exe
```

ou

```bash
./arvoreb.exe
```

---

## Observações

* O arquivo `pokemon_names.txt` deve permanecer na mesma pasta do executável gerado.
* Durante a execução, o programa solicitará a ordem da Árvore B.
* Após o carregamento dos dados, será exibido um menu com as opções de busca e inserção de Pokémons.
